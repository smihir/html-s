Internships

Cypress Semiconductor

Co-op Intern

Worked in Fall 2009 to develop reference designs for Blood Pressure Monitor using PSoC3 devices. The challenge here was to come up with a filter design and an algorithm which can accurately determine the Blood Pressure using the limited resources on a single PSoC3 chip. Our solution consisted of cascading an Instrumentation Amplifier and a Band Pass filter made using the analog blocks in PSoC. The signal was further smoothened before feeding to the BP calculating algorithm by using single pole IIR filters in the digital domain to reduce the 1/f noise which was found to result in inaccuracies.
